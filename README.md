# Cucumber Clicker (WPF)

Простой кликер, переписанный с WinForms на WPF.
Цель игры — накапливать закатки, покупать улучшения и достичь перерождения.

### Функционал
- Ручной клик
- Система покупки улучшений (рост цен)
- Пассивный доход (авто-кликер "Бабушка")
- Система Rebirth (открывается на 100 млн)

### Технологии
- C#
- WPF (XAML)
- .NET

### Запуск
1. Открыть WpfApp1.sln в Visual Studio
2. Собрать решение
3. Запустить
